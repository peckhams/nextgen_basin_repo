The USGS maintains a web page for each station/gauge with detailed
information for that location.  The URLs for these web pages end
with the USGS site number or ID, which may be 8 to 15 digits long.
Here is an example:
   https://waterdata.usgs.gov/nwis/inventory/?site_no=01206900

The USGS also maintains web pages for stations of the following
form that also end with the USGS site number or ID.
   https://waterdata.usgs.gov/monitoring-location/06827000

The USGS also maintains web pages that provide information for locations
based on their Hydrologic Unit Code or HUC.  Those URLs end with
the HUC number.  Here is an example:
   https://water.usgs.gov/lookup/getwatershed?10250001

Links to these example web pages are given in this folder.

You can also go to the USGS National Water Dashboard and click on a map
showing the USGS gages to get information for a given gage.
   https://dashboard.waterdata.usgs.gov/app/nwd/en/?aoi=default

Additional URLs for downloading data can be found in the subfolders of
the Data folder.
